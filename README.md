# bigstep-frontend
* React로 구현
* yarn으로 패키지 관리
* Typescript 적용

---

### frontend 서버 띄우기

1. yarn(javascript 패키지 매니저) 설치
```bash
$ curl -o- -L https://yarnpkg.com/install.sh | bash
또는
$ sudo apt-get install yarn
```

2. javascript 패키지 설치(처음과 패키지 주가됐을때만 수행하면됨)
```bash
$ yarn install
```

3. frontend 서버 실행
```bash
$ yarn start
```
